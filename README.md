# Simon Says Game 

A simple *Simon Says Memory Game* built using *HTML, CSS, and JavaScript*.  
The game challenges the player to remember and repeat an increasingly long sequence of colors. Each level adds a new color to the pattern, testing memory and concentration.

---

## Features
- Interactive color buttons with flash animations
- Increasing difficulty with each level
- User click feedback animation
- Score tracking by level
- High Score system
- Game Over screen with restart option
- Keyboard start control

---

## How to Play
1. Open the game in your browser.
2. Press *any key* to start.
3. Watch the color sequence flashed by the game.
4. Repeat the sequence by clicking the colored buttons in the same order.
5. Each correct round increases the level.
6. If you click the wrong color, the game ends and your score is displayed.

---

## Tech Stack
- *HTML* – Structure of the game
- *CSS* – Styling and animations
- *JavaScript* – Game logic and interactivity
